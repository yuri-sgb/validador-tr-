# Validador Inteligente de TR

Ferramenta para:

- Extrair ordem da tabela do PDF
- Reorganizar descritivo HTML conforme CATMAT
- Gerar:
  - HTML final pronto
  - HTML comparativo
  - Word (.docx)
  - Excel (.xlsx)

## Como usar

1. Envie o PDF do TR (com tabela).
2. Envie o HTML do descritivo.
3. Clique em "Processar Documento".
4. Baixe os arquivos gerados.

Desenvolvido para validação institucional de Termo de Referência.
